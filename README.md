# Activity Spark

A web application that helps users find random activity suggestions to combat boredom. Get inspired with activities across different categories like education, DIY, social, cooking, charity, and recreational.

## Installation
1. Clone the repository:

 `git clone git@github.com:safiabulle/activity-spark-project.git`

2. Install dependencies:

 npm install

## Usage 

To run the project, simply open the `index.html` file in your web browser.

1. **Home Page**: Click "I'm Bored" to get a random activity suggestion. Click "Save Activity" to add it to your favorites.
2. **Categories Page**: Select a category from the dropdown and click "Find Activity" to get a suggestion in that category.
3. **Favorites Page**: View your saved activities. Click "Remove" next to any activity to delete it from favorites.

## Features

- **Home Page**: Get random activity suggestions instantly
- **Categories Page**: Filter activities by specific interests (Education, Social, Recreational, DIY, Charity, Cooking)
- **Favorites Page**: Save your favorite activities and manage them with remove functionality
- **Responsive Design**: Works on desktop and mobile devices
- **Local Storage**: Persist favorite activities in the browser
